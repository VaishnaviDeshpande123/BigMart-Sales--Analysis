# BigMart Sales Analysis (SQL Project)

## Objective
Analyze BigMart sales data to identify key factors affecting sales and outlet performance.

## Tools Used
- SQL

## Key Analysis
- Data cleaning and preprocessing
- Sales analysis by item type and outlet
- Top and least selling products
- Outlet-wise and location-wise analysis

## Key Insights
- MRP significantly impacts sales performance
- Tier 3 outlets generate higher sales
- Supermarket Type 1 outlets contribute most revenue
- Dairy products show consistent demand

## Conclusion
This project helps understand sales patterns and supports data-driven business decisions.
